Parche temporal de compatibilidad de "Sorcerer Lulba 2" (Por DrWh0)
-------------------------------------------------------------------

He creado este parche para poder usar este pequeño juego homebrew
gratuito ya que los volcados existentes en internet se basan en una
copia defectuosa volcada por TOSEC.

Nombre del fichero TOSEC y checksum (CRC32) del disco defectuoso:

Nombre: "Lulba 2 (1988)(Compile)(JP).dsk" 
CRC32:  487BADEA

Este parche sustituye completamente a mi anterior parche IPS 
temporal que sólo pudo ejecutar los errores de ejecución.

En resumen:

El parche arregla el juego y se puede jugar y completar normalmente

Agradecimientos especiales:

que necesitaba para completar este parche
